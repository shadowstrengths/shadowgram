Which Light Are You?

Qual é a sua luz?

Shadowgram은 MBTI® 유형을 인지기능 축으로 분류한 8가지 고유 유형을 사용합니다. 각 유형은 주기능(빛)과 열등기능(그림자)의 축으로 구성됩니다.

Shadowgram uses 8 unique types defined by cognitive-function axes. Each type is shaped by its dominant function (Light) and inferior function (Shadow).

O Shadowgram usa 8 tipos únicos definidos por eixos de funções cognitivas. Cada tipo é moldado por sua função dominante (Luz) e função inferior (Sombra).
