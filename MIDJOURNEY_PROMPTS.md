## 완성 후 교체 방법

1. Midjourney에서 각 캐릭터 생성 (1:1 비율)
2. 배경 제거: remove.bg 또는 Photoshop
3. `share_images/` 폴더에 아래 파일명으로 저장:
   ```
   char_SPARK.png
   char_VISION.png
   char_STEADY.png
   char_PLAYER.png
   char_HARMONY.png
   char_SOUL.png
   char_LOGIC.png
   char_LEADER.png
   ```
4. Claude에게 "캐릭터 이미지를 hero/og에 합성해줘" 요청
   → 자동으로 기존 배경에 캐릭터를 오버레이
